## 🔒 安全特性
1. **文件类型验证** - 只允许图片文件
2. **文件大小限制** - 最大 10MB
3. **临时文件清理** - 处理完成后自动清理

## 📊 技术栈
- **后端**: Node.js + Express.js
- **前端**: HTML5 + CSS3 + JavaScript
- **文件上传**: Multer
- **API 调用**: Axios
- **背景移除**: Remove.bg API
- **部署**: 本地服务器 + 公网 IP

## 🚨 注意事项
1. **API 配额** - Remove.bg API 有免费额度限制
2. **网络延迟** - 图片上传和处理需要时间
3. **文件存储** - 处理后的文件保存在 public 目录
4. **服务器重启** - 服务重启后需要重新启动
